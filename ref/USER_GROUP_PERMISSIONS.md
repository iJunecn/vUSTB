## 关键权限差异总结

### super_admin 独有权限
- 设置用户组为 super_admin / admin
- 查看/设置用户积分
- 创建 admin 邀请码

### admin vs teacher
- **admin** 可访问完整管理后台（用户管理、系统设置、材质/角色/白名单管理）
- **teacher** 仅可：创建邀请码（含 teacher 类）、管理打印系统
- **teacher** 不可访问 `admin.py` 中的任何端点（依赖 `get_current_admin`）
- **teacher** 不可访问 admin 可见性的文件

### teacher 定位
teacher 角色的 `is_admin = False`，不属于管理员组。其权限集中在打印系统管理，是"打印管理员"角色，而非系统管理员。
